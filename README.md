# chs\_comments\_sentiment\_classfify

## **基于Python的中文评论情感分类系统**



简介：本项目采用传统机器学习方法构建的中文评论（短文本）情感分类基线模型。采用tfidf统计词频和svm支持向量机进行二分类。



语料：本项目采用微博公开语料库，github开源中文语料库。使用pre\_handle.py进行语料预处理，首先随机打乱语料并分割2/8比例分为80%训练集和20%验证集。对训练集原语料进行添加标签，并分为积极语料库和消极语料库。语料格式为“语料，0/1”的形式，以（逗号分割）csv文件保存，其中0代表消极，1代表积极。对验证集原语料进行添加标签并独立存放对应的0/1验证标签。以上文件存入对应文件夹。



模型训练：（版本python3.11）导入sklearn相关库。首先加载停用词，停用词来自哈工大开源停用词表。导入正向和负向语料训练库。采用pipeline方法对训练进行封装，先进行jieba分词，采用tfidf统计词频向量。之后导入svm支持向量机进行分类，并使用超参数调优。训练完成的模型文件用joblib保存。

后端封装：app.py使用后flask封装，并写入单条分析的api接口。



前端：采用vue+js的方式构建简单的前端，并写入多种功能如单条分析，批量分析（通过单条接口），历史记录等。
